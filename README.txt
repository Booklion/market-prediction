
# Market Prediction Application

This repository contains the code for my application.

**Table of Contents**

    1. Description
    2. Installation
    3. Usage
    4. Contributing
    5. License

**Description:** This project serves as a predictor for market fluctuations using a basic linear regression model.

**Installation:** To set up the project locally, follow these steps:

* Clone the repository:

        git clone https://github.com/Booklion/market-prediction.git

* Navigate to the project directory:
        
        cd market-prediction

**Usage:** The project structure is as follows:
1. main_script.py: The module that starts the application.
2. market-predictor/: Directory containing the Python files

**Contributing:**

*Branches*
* master: Main branch containing the stable version of the application.
* feature/add-about-page: Branch for adding the ’About Me’ section to the website.
